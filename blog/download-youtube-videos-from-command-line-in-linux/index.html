Скачать Видео с YouTube из Командной Строки в Linux

09-06-2017  /  shellhacks.com

Мы можем скачивать видеофайлы с YouTube из командной строки в Linux с помощью программы youtube-dl.

Это небольшая консольная утилита, которая умеет скачивать видео с youtube.com и еще ряда сайтов.

В данной статье вы найдете информацию о том, как в зависимости от используемого Linux дистрибутива, установить загрузчик для YouTube.

Также, вы узнаете как из командной строки в Linux загрузить видео, канал или плейлист с YouTube в определенном качестве и формате.
